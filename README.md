# ProphetBoost-A-Hybrid-Pipeline-for-Accurate-and-Transparent-Electricity-Load-Forecasting
Implementation of ProphetBoost, a hybrid electricity load forecasting framework combining Prophet decomposition, stability-guided feature selection, and XGBoost. Includes preprocessing pipeline, benchmark models, hyperparameter tuning, statistical tests, and reproducible experiments.
